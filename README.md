* ### 基本功能：

1. 拍照或相册选择发布动态，支持语音
2. 动态广场点赞评论，支持语音评论
3. 用户之间关注
4. 用户之间私聊，支持文本、图片、语音形式信息
5. 。。。其他玩法待开发

* ### 技术特色

1. DataBinding绑定数据，降低乏味操作并且无反射，一定程度优化性能
2. MVP结构，思路清晰，解耦
3. RxJava+Retrofit 网络框架，稳定高效
4. java web + 阿里云 + Leancloud 后台解决方案
5. 热更新，修改更新更便捷
6. face++ 机器学习应用
7. Android自定义相机,音频流及时处理,方便后期处理
